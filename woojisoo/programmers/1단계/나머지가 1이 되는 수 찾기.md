# 나머지가 1이 되는 수 찾기

## 풀이

## 코드

```javascript
function solution(n) {
  for (i = 0; i < n; i++) {
    if (n % i === 1) {
      return i;
    }
  }
}
```

## 다른 사람의 풀이

```javascript
function solution(n, x = 1) {
  while (x++) {
    if (n % x === 1) {
      return x;
    }
  }
}
```

while문 보다는 for문이 더 익숙했는데 앞으로 while문을 많이 써보자.
